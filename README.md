# AluraGeek
AluraGeek é um challenge proposto pelo curso Front End G7 - ONE
AluraGeek é um site de comércio eletrônico voltado para fãs de cultura geek e nerd. O projeto apresenta uma interface atraente,
 com um layout responsivo que permite aos usuários explorar uma variedade de produtos, como jogos, figuras de ação e muito mais.
